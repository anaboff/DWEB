COMO UTILIZAR OS TIPOS DE HIPERLINKS

Hiperlinks Externos

Os hyperlinks externos são links para páginas externas, ou seja, que está fora da página em questão, é uma página a parte. Para adicionarmos um link em nosso código
utilizamos a tag HREF (< a href=”link para a página”>Nome da Página< /a>).Essa tag pode ser usada para vários tipos de links como links de aúdio, vídeo, texto e imagem.

Temos a opção de colocar a propriedade TARGET logo após o link, são duas opções, uma é target=”_blanck” para abrir em outra página, que ficaria assim
(< a href=”link para a página” target=”_blanck”>Nome da página< /a>) . A outra opção é target=”_self” para abrir na mesma página, que ficaria assim
(< a href=”link para a página” target=”_self”>Nome da página< /a>).

Propriedade REL

A propriedade REL (rel="external") informa se o link que aparece na página é externo, ela é utilizada também logo após a tag HREF que ficaria assim
(< a href="link para a página" target="_black" rel="external">Nome da página< /a>).

HREFLANG

HREFLANG (hreflang="linguagem do site") é utilizado para definir qual a linguagem que o site é, para caso a pesssoa prefira, ele faz a tradução automática. Então ficaria assim
(< a href="link da página" hreflang="a linguagem do site" target="_blanck" rel="external">Nome da página< /a>).

Hyperlinks Internos

Os hyperlinks internos são links que redirecionam para uma seção ou parágrafo da página atual, usados muitas vezes em páginas com textos muito longos. Para colocarmos os
hyperlinks internos você vai utilizar a mesma tag HREF com algumas mudanças.No lugar do link da página você vai colocar #referencia, ficará assim
< a href="#referencia">Conteúdo refenciado< /a>, então na próxima linha você colocará < a name="referencia">< /a>, por fim ficará dessa forma:

  < a href="#referencia">Conteúdo referenciado< /a>
  < a name="referencia">< /a>

Hyperlinks Locais

Os hyperlinks locais são links que apontam para páginas de um mesmo site. Para fazer isso continuaremos utilizando a tag HREF, mas no lugar do link da página colocamos o
nome do arquivo html, dessa forma < a href="arquivo.html">Conteúdo referenciado< /a>
