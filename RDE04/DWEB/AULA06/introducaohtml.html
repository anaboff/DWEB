HTML – Hypertext Markup Language

A linguagem HTML foi criada por Tim Tim Berners-Lee (o mesmo criador do World-Wide-Web (www) em 1990 para que se pudesse compartilhar docu-
mentos científicos entre os acadêmicos. Ele utilizou como base a linguagem de marcação SMGL.

O HTML é uma linguagem de marcação, sua intenção não é criar um software, mas sim um conjunto de regras para que se possa estruturar a página web
e arrumar componentes visuais. Mesmo não sendo considerada uma linguagem de programação, o HTML possui regras de sintaxe e palavras reservadas
de modo semelhante.

A linguagem é baseada em tags, para publicações de conteúdo em hipertexto como texto, imagem, áudio e vídeo, e que podem ser vinculados através de
hyperlinks.

TAGS

As tags definem ao navegador a marcação de conteúdo. As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.
As tags podempossuir atributos, ou seja,são parâmetros que definem algum tipo de informação extra ao navegador.

Para definir uma tag, coloca-se o nome da mesma entre parênteses angulares (<,>), e no fim a mesma coisa, porém antes do nome da tag, coloca-se uma
barra na frente, no meio das tag de início e tag de fim adiciona-se o conteúdo.

Sintaxe:
    < nome_tag> conteúdo < /nome_tag>

As tags também podem possuir atributos, que são os parâmetros que definem algum tipo de informação extra ao navegador.

Sintaxe:
    < nome_tag> conteúdo < /nome_tag>

Estrutura HTML

Acima está a estrutura básica do HTML, porém ainda é necessário defninir algumas tags para que o documento esteja completo.Um deles é o DTD-Document
Type Declaration, que parece ser uma tag HTML mas sim uma declaração de tipo de documento, que serve para o navegador entender que no documento esta-
mos utilizando o HTML 5

Sintaxe:
    < !DOCTYPE html >

Depois colocaremos o parâmetro de linguagem, que ficará dentro da tag HTMl, para que o navegador saiba qual língua o documento está.

Sintaxe:
    < html lang=”pt-BR” >

Dentro da tag HEAD, colaremos mais algumas definições para que o navegador possa ajustar-se ao que o queremos. As tags utilizadas nessa parte serão as de me-
tadados e a primeira que é e que informa ao navegador que será utilizada um tipo de codificação específica. O código UTF-8 é um código multibyte, de codificação
binária que representa qualquer carácter.

A próxima tag metadado é a e informa ao navegador qual a tela que estamos utilizando. Como hoje é possível acessar as páginas web por celulares, computadores esmart tvs, a tag vai definir as dimensões e diensionamento da página

Sixtaxe:
    < meta charset=”UTF-8” >
    < meta name=”viewport content=”widht, initial-scale=1.0” >

Marcação de quebra de linha

A tag que quebra de linha, e produzem uma quebra de linha e um texto. Esta tag, diferente das outra, não tem uma tag de fechamento. Ela é muito útil
ao escrever poemas e endereços, ou textos que sejam necessário quebra de linhas.

Sintaxe:
    Essa é uma < br > quebra de linha
    Essa é uma
quebra de linha
