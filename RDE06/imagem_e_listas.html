Imagem

Imagem é uma representação visual de uma pessoa, objeto ou paisagem. Quanto trata-se de uma imagem em termos computacionais, podemos dizer que é uma
imagem digital e será a representação de uma imagem bidimensional usando números binários codificados de modo a permitir seu armazenamento, transferência,
impressão ou reprodução, e seu processamento por meios eletrônicos.

Formatos de imagens

Formatos de imagem são padrões de codificação, compreensão e definição que determinam o tipo de arquivo gráfico, tais como, .BMP, .TIFF, .GIF, .SVG, .ICO,
.JPEG, .PNG dentre outros. As diferenças entre as extensões de arquivos é que elas vão compactar o arquivo, deixando-o maior ou menor em números de bytes,
megabytes, ou kilobytes.

Marcação de Listas

As tags de listas definem: listas não-ordenadas, listas ordenadas, e lista detalhada.

Listas não-ordenadas:

Sintaxe:
  < ul >
     < li > Item 1 < /li >
    < li > Item 2 < /li >
    < li > Item 3 < /li >
  < /ul>

Sintaxe:

  • Item 1
  • Item 2
  • Item 3

Listas ordenadas:

Sintaxe:
  < ol type=”elemento”>
    < li > Item 1 < /li >
    < li > Item 2 < /li >
    < li > Item 3 < /li >
  < /ol>

Sintaxe:

  1. Item 1
  2. Item 2
  3. Item 3

    4.   

Listas detalhadas:

Sintaxe:
  < dl>
    < li > Item 1 < /li >
    < li > Item 2 < /li >
    < li > Item 3 < /li >
  < /dl>

Sintaxe:

  • Item 1
  • Item 2
  • Item 3
